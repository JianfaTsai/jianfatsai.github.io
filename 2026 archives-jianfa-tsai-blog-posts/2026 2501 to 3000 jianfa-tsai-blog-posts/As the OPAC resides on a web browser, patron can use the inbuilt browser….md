However, this requires a higher level of tech-savviness or IT knowledge. Some users are unaware of the keyboard shortcut CMD + plus key or CMD + minus key to zoom in/out.   
   
The library website has a yellow floating accessibility button on the right-hand side of the web page that does not disappear as you scroll the web page vertically up or down. The accessibility button allows users to adjust the webpage text font size, formatting, contrast, grayscale, and orientation (highlight links and reading mask).
