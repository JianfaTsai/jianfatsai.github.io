How do you setup Grammarly for Microsoft Words on macOS?  
  
AI Analysis: Setting up **Grammarly** (Grammarly) — a popular AI-powered writing assistant that checks grammar, spelling, style, tone, and more — in **Microsoft Word** (Microsoft Word) on macOS has changed. The dedicated **Grammarly for Microsoft Word** (Grammarly for Microsoft Word) add-in was officially discontinued by Grammarly.  
  
**ELI5 in a single sentence:** Grammarly used to have a special button inside Word on Mac, but they stopped that; now you install their main Mac app, and it works inside Word like a helpful friend watching over your typing.

### Current Recommended Setup (as of 2026)  
Grammarly recommends installing **Grammarly for Mac** (Grammarly for Mac) — their desktop application that integrates system-wide, including with **Microsoft Word** (Microsoft Word) on macOS 10.15 (Catalina) or newer. This provides real-time suggestions, a sidebar panel, and advanced features like tone detection and plagiarism checking directly in Word (and other apps).  
  
#### Step-by-Step Installation  
1. **Create or Log In to Your Grammarly Account**    
   Go to grammarly.com and sign up (free tier available) or log in. A Premium or higher subscription unlocks more features.  
  
2. **Download Grammarly for Mac**    
   Visit the official page: https://www.grammarly.com/desktop/mac    
   Click the download button. The file is usually a .dmg installer.  
  
3. **Install the App**    
   - Open the downloaded .dmg file.    
   - Drag the Grammarly icon to your Applications folder (as prompted).    
   - Eject the disk image and launch Grammarly from Applications or Launchpad.  
  
4. **Set Up and Log In**    
   - Open Grammarly for Mac.    
   - Sign in with your Grammarly account.    
   - Grant any necessary permissions (e.g., Accessibility for system-wide functionality and access to Word documents).    
   - Grammarly will run in the background and show an icon in the macOS menu bar (top-right).  
  
5. **Enable in Microsoft Word**    
   - Open Microsoft Word (version 2016 or later recommended; Microsoft 365 is ideal).    
   - Start typing or open a document.    
   - Grammarly should automatically detect text in Word and show:    
     - Colored underlines for suggestions.    
     - A **Grammarly tab/panel** on the right side of the document for detailed checks.    
   - If it doesn't appear:    
     - Ensure Grammarly for Mac is running (check menu bar).    
     - Switch Word view to **Print Layout** (View > Print Layout) — Grammarly may not work in Draft, Outline, or Web Layout.    
     - Restart Word or your Mac.  
  
**Note on the Old Add-in:** Previously, users could go to Insert > Get Add-ins (or My Add-ins) in Word, search for "Grammarly," and install it directly. This is no longer supported on macOS due to the discontinuation. Attempting it may show errors or no results.  
  
### How It Works in Word Once Set Up  
- As you type, Grammarly highlights issues (grammar, clarity, engagement, etc.).    
- Click underlined words for quick fixes or open the full Grammarly sidebar for in-depth suggestions.    
- Advanced features (available on paid plans): rewrite with AI, tone adjustment, plagiarism detection (limited to Word and Pages on Mac), and generative AI prompts.    
- It works offline for basic checks but needs internet for full AI features and syncing.  
  
### Edge Cases and Related Considerations  
- **Apple Silicon (M1/M2/M3/M4) Macs:** Grammarly for Mac is native and runs efficiently.    
- **Microsoft 365 vs. One-Time Purchase Word:** Both should work, but ensure Word is updated via the Mac App Store or Microsoft AutoUpdate.    
- **Performance:** On large documents, real-time checking might slightly slow things down — you can pause Grammarly via the menu bar icon if needed.    
- **Multiple Apps:** The desktop app also works in Pages, email clients, browsers (via extension), and more — no separate installs required.    
- **Permissions Issues:** If suggestions don't appear, go to System Settings > Privacy & Security > Accessibility and ensure Grammarly is enabled. Restart apps after changes.    
- **Free vs. Premium:** Free version covers core grammar/spelling. Premium adds style, tone, fluency, and AI tools.    
- **Uninstall Old Add-in (if present):** If you have remnants of the discontinued add-in, remove it via Word's Add-ins manager to avoid conflicts.    
- **Alternatives if Issues Persist:** Use Grammarly's web editor (app.grammarly.com) and copy-paste into Word, or consider built-in Microsoft Editor (in Word's Review tab) as a lighter option.
