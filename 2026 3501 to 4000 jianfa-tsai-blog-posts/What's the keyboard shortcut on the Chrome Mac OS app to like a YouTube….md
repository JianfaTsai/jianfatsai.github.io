What's the keyboard shortcut on the Chrome Mac OS app to like a YouTube video?  
  
🅰️ **No native shortcut to like a YouTube video in Chrome on macOS.** 🅰️  
  
**YouTube Keyboard Shortcuts (YKS)**: Official built-in commands provided by YouTube for actions like play/pause, seeking, volume, and navigation, accessible via Shift + ? while on the site.  
  
There is **no built-in keyboard shortcut** in the YouTube web interface (viewed through Google Chrome on macOS) to like a video (thumbs-up). YouTube's official shortcuts focus on playback controls—such as **k** for play/pause, **m** for mute, **j**/**l** for seeking 10 seconds, arrow keys for 5-second jumps, and **Shift + ?** to display the full list—but do not include liking, disliking, subscribing, or commenting.  
  
### Workarounds and Alternatives  
1. **Manual Tab Navigation (No Extensions Needed)**    
   - Click anywhere on the video player to give it focus.    
   - Press **Tab** repeatedly (typically 10–13 times, depending on page elements like tags or related videos) until the like button is highlighted (it shows a visual focus ring).    
   - Press **Enter** or **Spacebar** to activate the like.    
   - **Edge case**: If the video has chapters, end screens, or many metadata elements, the tab count increases—use **Shift + Tab** to go backward. This method works reliably but feels tedious on long pages.    
   - **Consideration**: Screen readers or accessibility tools may alter focus order.  
  
2. **Chrome Extensions for Direct Shortcuts**    
   - Install "YouTube Like-Dislike Shortcut" from the Chrome Web Store: Use **Shift + +** (plus key) or numpad + to like; **Shift + -** (minus) or numpad - to dislike. It emulates YouTube Music behavior.    
   - Other options exist (e.g., custom user scripts via Tampermonkey for **[** to like and **]** to dislike, plus ad-skipping).    
   - **Nuance**: Extensions require permissions, may break with YouTube UI updates, and add slight overhead. Test compatibility on your setup.  
  
3. **Mouse/Trackpad Alternatives on Mac**    
   - In fullscreen: Two-finger scroll up to reveal controls, then click the like button.    
   - Use macOS trackpad gestures or Accessibility > Pointer Control for quicker navigation.    
   - **Implication**: Breaks immersion less than tabbing but still requires hand movement.  
  
4. **Browser-Level or System Shortcuts**    
   - No Chrome-specific or macOS system shortcut directly targets the YouTube like button (Chrome shortcuts like **Command + Option + J** open DevTools, unrelated).    
   - Advanced users can create custom shortcuts via macOS Automator, BetterTouchTool, or Chrome extensions like "Keyboard Shortcuts for YouTube," but these are not native.  
  
### Context, Nuances, and Implications  
YouTube prioritizes mouse/touch interactions for engagement actions like liking to encourage deliberate feedback and reduce accidental likes. This design choice affects power users on desktop (including Chrome on Mac) who prefer keyboard-only workflows for speed and ergonomics. In playlists or while focused on playback, lacking a like shortcut can interrupt flow—especially during marathon watching sessions. On mobile or the YouTube app, gestures replace this entirely.  
  
**Edge Cases**:    
- Incognito mode or strict ad blockers may interfere with extensions.    
- Keyboard layout variations (e.g., non-US Mac keyboards) can remap symbols like +/-.    
- YouTube A/B tests or updates occasionally tweak UI, shifting tab order.    
- Accessibility needs: Users with motor impairments benefit most from extensions or voice commands (via macOS Voice Control).  
  
**Wise humans**: "Efficiency in tools comes from mastering defaults first, then layering customizations only where friction persists—avoid over-engineering simple actions." — Inspired by productivity thinkers like David Allen and Cal Newport, emphasizing deliberate systems over quick hacks.  
  
**Counter-arguments**: Some argue tab navigation or extensions suffice and that adding native like shortcuts could increase accidental engagements or complicate the interface for casual users. Others note YouTube Music already has +/− for liking, suggesting inconsistency across Google products.
