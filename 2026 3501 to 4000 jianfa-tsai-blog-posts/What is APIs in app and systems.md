# What is APIs in app and systems?  
  
**APIs (Application Programming Interfaces)** are like **bridges** between apps and systems.  
  
### Simple Explanation:  
An **API** lets one app or system **talk to another** and **share data or functions** without you seeing the inside details.  
  
### Real-life Examples:  
- **Weather app**: Uses an API from a weather service to get current temperature.  
- **Google Maps in Uber**: Uber app calls Google Maps API to show location and routes.  
- **Login with Google**: Your app uses Google’s API to let users sign in without creating a new account.  
- **Payment**: Apps use Stripe or PayPal API to process credit card payments safely.  
  
### How it Works (Super Simple):  
1. App A **asks** (sends a request) to App B via API.  
2. App B **answers** (sends back data).  
3. Done — no need to know how App B works inside.  
  
**In short**: APIs make apps work together smoothly, like Lego blocks connecting.
