# Would the cheapest multi-monitor setup for large screen estate be to buy three of the cheapest Mac minis to connect to each other and connect nine ARZOPA 2.5K Portable Monitor 16" 2560x1600 HDR Kickstand Display, 500 nits 8-bit QHD USB-C HDMI Laptop Monitors to a single Apple Magic Keyboard with Numpad and Apple Magic Mouse? Is the above workflow feasible? What’s the maximum number of Mac Minis that can be connected to each other? What’s the maximum number of portable monitors that can be connected to each other for the above setup?  
  
**No. Not the cheapest for unified large screen estate.**    
**Workflow feasible: yes**—3 M4 Mac Minis (each drives 3 ARZOPA monitors natively via TB/HDMI), networked with Universal Control for one Magic Keyboard/Mouse to control all seamlessly (cursor/clipboard flows across 3 independent macOS desktops).  
  
**Max Mac Minis connected (for shared input):** 3 (Apple Universal Control limit: 1 primary + 2 others).  
  
**Max portable monitors connected (in this setup):** 9 (3 per Mini; ARZOPA models have single USB-C/HDMI input only—no daisy-chaining).
