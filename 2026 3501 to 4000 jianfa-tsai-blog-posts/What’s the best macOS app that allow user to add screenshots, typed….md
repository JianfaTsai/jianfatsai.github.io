# What’s the best macOS app that allow user to add screenshots, typed text, Apple Pencil drawn drawings, and screen recording files?  
  
**CleanShot X** is the best dedicated macOS app for this. It excels at capturing screenshots (including scrolling), adding typed text/annotations, drawing (shapes, arrows, freehand highlights), and screen recordings—all with a powerful built-in editor.  
  
You can:  
- Take area/window/fullscreen/scrolling screenshots.  
- Annotate instantly: add text, draw freely, highlight, blur, or use shapes.  
- Record screen (with audio, mic, or camera) and export as video/GIF.  
- Combine multiple captures, use OCR for text extraction, and quickly share or save.  
  
It's fast, native-feeling, and highly praised for workflow efficiency on Mac. Paid (one-time or via Setapp), with a trial available.  
  
### Free/Built-in Alternative: Apple Notes  
For a no-cost option, use **Apple Notes** (pre-installed):  
- Drag in screenshots or screen recordings (use built-in Shift+Cmd+5 for capture).  
- Type text directly.  
- Markup images/PDFs with drawings, text, shapes, or highlights (Markup tool).  
- Add files easily and keep everything in one note.  
  
It handles most needs simply, with iCloud sync, but lacks CleanShot's advanced capture/editing polish.  
  
### Other Options  
- **Shottr** or similar lightweight tools: Great for quick screenshot annotation + drawings, but weaker on recordings.  
- Note apps like **GoodNotes** or **Notability** (Mac versions): Strong for typed/handwritten text + images/drawings in notebooks, with audio recording (not full screen video). Better if you're mixing handwriting heavily.  
- Built-in Preview: Quick markup on existing screenshots/files.  
  
**Recommendation**: Start with **CleanShot X** for seamless capture + editing in one tool. If you prefer an all-in-one notebook for organizing mixed content, try Apple Notes first—it's free and sufficient for most. Test via App Store or their sites.
