# Chrome Tab Navigation Extension Recommendation

### Jianfa Tsai's Input
What's a Chrome macOS extension where users press keyboard shortcuts and can move to the last previously accessed browser tab (similar to the web browser forward and backward page feature)?

### SuperGrok AI Analysis
The "Previous Tab" Chrome extension enables keyboard shortcuts like Ctrl+Q to switch to the last accessed tab on macOS, mimicking browser history navigation (Previous Tab, 2026).
