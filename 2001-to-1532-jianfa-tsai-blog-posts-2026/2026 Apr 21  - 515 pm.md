On macOS, how do I turn off the settings to prevent the mouse pointer selecting or activating the app displayed on the CMD+TAB dock? The mouse pointer frequently cause me to accidentally trigger open the wrong app.   
  
**macOS has no built-in setting to disable mouse hover/selection in the native Cmd+Tab app switcher.**  
**macOS has no built-in setting to disable mouse hover/selection in the native Cmd+Tab app switcher.**  
The best fix is to replace it with the free, open-source **AltTab** app:  
1. Download and install AltTab from ++[https://alt-tab-macos.github.io/](https://alt-tab-macos.github.io/)++  
2. Open **AltTab → Preferences**.  
3. Uncheck **“Also select windows using: Mouse Hover”**.  
4. (Optional but recommended) Set its trigger to **Cmd+Tab** so it fully replaces the native switcher.  
This stops the mouse from accidentally activating the wrong app while keeping keyboard-only switching.
