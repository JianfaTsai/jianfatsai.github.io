Takes less than 5 mins to set up.  
Install AutoHotkey app [ ++[https://www.autohotkey.com](https://www.autohotkey.com/)++ ] on your Windows laptop.  
It’s safe. Simply paste website links into Norton anti-virus checker here [ ++[https://safeweb.norton.com/](https://safeweb.norton.com/)++ ]  
  
Right click anywhere on your desktop to open a pop up menu. Hover your mice pointer over “New” which opens another pop up menu and click on “AutoHotkey Script”.  
![Attachment.png](Attachments/B643935F-432B-44F9-AD96-BBABE87BA300.png)  
This will create a script on your desktop.  
![Attachment.png](Attachments/D3FBDA99-EDB8-44C5-9F7F-C01FC6B42D3E.png)  
Name the script to your preferred titled e.g. “text helper”. Right click on the script icon and click “Edit Script”.  
It will open a notepad text file window.  
Ignore existing codes in the window and enter the below code in the white space of the window.  
:*:btw::by the way{Enter}  
The above is a sample code when you type “btw” it auto expands into “by the way” in any app or textfield. Important: This code only applies to plain English alphabetical and numbers only text free of any special characters e.g. # symbol.  
Simply replace “btw” without quotation marks as the shortcut phrase to your desired shortcut phrase and the expanded text “by the way” without quotation marks to your desired full bodied text. Keep the rest of the code as is.  
If you want to insert special characters into your expanded text snippet for social media.  
Change the # symbol in your sentence to the code +3, e.g.  
:*:ccode::+3computerscience +3programming{Enter}  
The +3 is for the # symbol above the number 3 on the keyboard.  
So when I type “ccode”, the “ccode” text will be replaced by #computerscience #programming  
![Attachment.png](Attachments/DB1117A7-D31C-420C-AC78-8E168646DC59.png)  
You can use this for quick insertion of emails, dates, and a plethora of other uses.  
For more details visit the AutoHotkey forums and help documentation here for tutorial for beginners and list of keys: [++[https://www.autohotkey.com/docs/Tutorial.htm](https://www.autohotkey.com/docs/Tutorial.htm)++]  
You can also google your questions e.g. “AutoHotkey # symbol” to that effect.  
You can add multiple text snippets into a single hotkey script file in separate lines e.g.  
:*:btw::by the way{Enter}  
:*:ccode::+3computerscience +3programming{Enter}  
Always remember to save the script and double click to run it again to ensure that the outcome that is currently produced is based on the new edits and not on the old codes.  
Finally, you can automate the script to auto run upon each computer bootup by  
1. Right click on the script file to create a shortcut file  
2. Hit the “Win” key  
3. Type “run” to open the “Run” app  
4. Type “shell:startup” without the quotation marks to open open the system startup folder  
5. Cut the script’s shortcut file and paste into the startup folder  
6. You can do this for any other files or apps you want the system to auto open upon log in
